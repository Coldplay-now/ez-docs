## Syntax Highlighting

ezdoc uses [Shiki](https://shiki.style) as its syntax highlighting engine, generating precise token-level coloring at build time with support for 100+ programming languages. Light and dark themes use independent color schemes (github-light / github-dark) that switch automatically with the theme.

### Python

```python
from pathlib import Path
from typing import AsyncIterator
import asyncio

async def watch_docs(directory: Path) -> AsyncIterator[Path]:
    """Watch a directory for markdown file changes."""
    seen: dict[Path, float] = {}

    while True:
        for path in directory.rglob("*.md"):
            mtime = path.stat().st_mtime
            if path not in seen or seen[path] < mtime:
                seen[path] = mtime
                yield path

        await asyncio.sleep(1.0)

async def main():
    async for changed in watch_docs(Path("./docs")):
        print(f"Changed: {changed}")
```

    AVG(v.reading_time_seconds) AS avg_reading_time

## Inline Code

Inline code is wrapped in backticks: config file `ezdoc.config.ts`, command `pnpm build`, type `Record<string, unknown>`, etc. Inline code automatically applies a monospace font and background color to visually distinguish it from body text.
