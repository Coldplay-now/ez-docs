## Text Formatting

| Syntax | Result |
|--------|--------|
| `**bold**` | **Bold text** |
| `*italic*` | *Italic text* |
| `~~strikethrough~~` | ~~Strikethrough text~~ |
| `` `inline code` `` | `inline code` |
| `**_bold italic_**` | **_Bold italic text_** |

## Links

- External links: [GitHub](https://github.com) automatically opens in a new tab
- Internal links: [Back to Getting Started](/docs/en/getting-started) uses client-side routing
- Anchor links: [Jump to Tables section](#tables)

## Blockquotes

> This is a blockquote. Blockquotes are useful for emphasizing important notes or citing others.
>
> Blockquotes can contain multiple paragraphs and nest other formatting like **bold** and `code`.

## Code

### Inline Code

Use `pnpm install` to install dependencies. The configuration file is located in `ezdoc.config.ts`.

### Code Blocks

```bash
# Clone the repository
git clone https://github.com/Coldplay-now/ezdoc.git
cd ezdoc

# Install dependencies and start
pnpm install
pnpm dev
```
